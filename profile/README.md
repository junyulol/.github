<p align="center">易之四象 悟其本元</p>

<p>易象元，千变万化的起源，易之四象，悟其本元。</p>
<p>四季、星河、天地、万物，皆在自然流转、悄然更替。</p>
<p>然，万象有初，是为本元。</p>
<blockquote>#易象元 即任自然变迁、古树难寻，亦为这朗朗乾坤间灵秀的一芽两叶，跨越万水千山，不论大势所向，始终如初，悉心呵护这天地精华呈上一杯可品出自然，获得康健的茶汤~</blockquote>
